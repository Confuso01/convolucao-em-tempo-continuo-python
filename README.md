# convolucao-em-tempo-continuo-python
Aplicação em Python para visualização da convolução de sinais em tempo contínuo, utilizando NumPy, SciPy, Matplotlib e Tkinter. Permite definir funções matemáticas, intervalos, domínio, número de pontos e escolher o método de convolução. Inclui interface gráfica, ajuda integrada e uma biblioteca de exemplos prontos para estudo e ensino.
Bibliotecas para instalar:
pip install numpy scipy matplotlib
