# convolucao-em-tempo-continuo-python
Aplicação em Python para visualização da convolução de sinais em tempo contínuo, utilizando NumPy, SciPy, Matplotlib e Tkinter. Permite definir funções matemáticas, intervalos, domínio, número de pontos e escolher o método de convolução. Inclui interface gráfica, ajuda integrada e uma biblioteca de exemplos prontos para estudo e ensino.
Bibliotecas para instalar:
pip install numpy scipy matplotlib

Convolução de Sinais Contínuos – Wiki

Esta aplicação permite visualizar e estudar a convolução de sinais contínuos usando uma interface gráfica feita em Python (Tkinter). O usuário pode definir duas funções f(t) e g(t), configurar seus intervalos de existência, escolher o domínio de visualização e calcular a convolução usando NumPy ou SciPy.

A interface exibe três gráficos:

f(t)

g(t)

Convolução f * g

Também há uma área de ajuda integrada explicando como escrever funções com NumPy, como funcionam os intervalos e como escolher o método de convolução.

O programa inclui ainda uma janela com diversos exemplos prontos, permitindo carregar rapidamente sinais clássicos como pulsos, exponenciais causais, sinais modulados, respostas de circuitos e outros casos reais.

O objetivo principal é fornecer uma ferramenta simples e didática para disciplinas de processamento de sinais, sistemas lineares, eletrônica e áreas relacionadas.
